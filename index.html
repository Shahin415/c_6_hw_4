Display property are 3 types

  • Inline element=It dosen't allow other element beside him Ex:h1,p,div,table etc
  • Block Level=It is allow other element beside him Ex:a,img,span etc
  • Inline Block Element=It is allow other element beside him but dosen't allow in top and bottom side

    • Span=Element which is used to color a part of a text

  • Semantic Element=Clearly defines its content
  • Non-Semantic Element =It dosen't clearly defines its content
  • Margin =used to move a element form other element with space outside of border
  • Padding = properties are used to generate space around an element's content, inside of any defined borders.
